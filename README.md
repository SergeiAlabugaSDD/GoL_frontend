# GoL_Frontend

It's a client-side of application, implementing known "Game of Life"

<!-- See deployed version on [heroku.com][df2] -->

Tools:

1. react
2. react-router
3. React D'n'D
4. PostCSS(stage 0)
5. redux
6. redux-saga
7. eslint-airbnb

<!-- Powered by [DiceBear Avatars][df1] -->

### To run project locally:

```
npm i
```

and

```
npm run start
```

#### Or, if you have Yarn

```
yarn
```

then

```
yarn start
```
